How It Works

funneljs works by using a broad collection of elements which are then subjected to filters to get/exclude elements.

  • General Overview:
    • Step 1 — Get elements from source points:
      • f("#source_point_id", "#source_point_id_N")
    • Step 2 — Chain filters to get/exclude elements:
      • .filter1().filter2().filterN()
    • Step 3 — Finally, return element(s) via:
      • getStack or getElement:
    • Example — f("#aside:all").tags("span", "div").getStack();
    • Explanation — Query uses the element with the ID of aside as its source point. The inline :all means to grab ALL its descendants. The tags() filter is then used on the elements collection to only get elements with the tags of span or div. Finally, the filtered collection is returned for use as an Array with getStack().

Instance Creation

  • sourcePoint (String|HTMLElement, Required) The source point to use.
    • Note: n amount of sourcePoints may be passed.
  • Returns instance.

Note: Using the new keyword is not necessary. The library will make sure to use it for when when you don't.

// this...
var query = new Funnel("#container", "#sidebar");
// is the same as this
var query = Funnel("#container", "#sidebar");

What's A Source Point?

A source point is just an element. This element is used to grab all its descendants to build the collection of elements wanted to filter. Querying the DOM is an expensive task as it searches the entire DOM for your wanted elements. Rather than search the entire DOM, this method pin-points its search on the descendants of selected source point elements.

Source Point Comparisons

// this...
f("#aside"); 
// is essentially...
document.getElementById("aside");
// this...
f("#aside:all"); 
// or...
f("#aside").all();
// is essentially this...
document.getElementById("aside").getElementsByTagName("*");
// this...
f("#aside", "#footer:all");
// is the essentially both this...
document.getElementById("aside");
// and this...
document.getElementById("footer").getElementsByTagName("*");
// this...
f("#aside:all", "#footer:all");
// is the essentially both this...
document.getElementById("aside").getElementsByTagName("*");
// and this...
document.getElementById("footer").getElementsByTagName("*");

The Stack

The initial elements that are passed to the Funnel instance make up the first stack. Each method (filter) used after that will create an additional element stack. Therefore, the getStack method will always return a stack, or element collection. By default it returns the last stack but can return any of the previous stacks. The getElement method will always return the first element of the last stack.

// For example, the following contains 2 stacks...
f("#red", "#green").attrs("[id=red]") 

// first stack  (index:0) --> ["<#red>", "<#green>"]
// second stack (index:1) --> ["<#red>"]

// Therefore...

f("#red", "#green").attrs("[id=red]").getStack() 
// will return the last (most current) stack --> ["<#red>"]

Instance Methods

➜ instance.all — Gets children and descendants of elements in last stack.

  • No Parameters
  • Returns instance.
// can be combined with source element
var query = f("#aside:all");
// or used as a chained method
var query = f("#aside").all();

➜ instance.attrs(attribute, attributeN...) — Gets elements matching all supplied attributes.

  • attribute (String, Required)
    • Note: n amount of attributes may be passed.
  • Returns instance.
// Example 1: gets elements that HAVE a class attribute
query.attrs("[class]");

// Example 2: gets elements that DO NOT have a class attribute
query.attrs("[!class]");

// Example 3: gets elements with a type attribute AND value equal to text
query.attrs("[type=text]");

➜ instance.children — Gets element's children.

  • No Parameters
  • Returns instance.
query.children();

➜ instance.classes(className) — Gets elements matching any of the supplied classes.

  • className (String, Required)
    • Note: n amount of classNames may be passed.
  • Returns instance.
// Example 1: gets elements with the classes active & nav-item
query.classes("active", "nav-item");

// Example 2: gets everything but elements with the classes active and nav-item
query.classes("!active", "!nav-item");

➜ instance.form(inputType) — Selector.attr shorthand. (List of possible input types here and here)

  • inputType (String, Required)
    • Note: n amount of inputTypes may be passed.
  • Returns instance.
// Example 1: gets elements that have a type attribute and value text => [type=text]
query.form(":text");
// shorthand for...
query.attrs("[type=text]");

➜ instance.next — Gets the next element sibling of elements in last stack.

  • No Parameters
  • Returns instance.

Note: If no element exists null is returned.

f("#aside1").next(); // <#aside2>

➜ instance.only(indices) — Filters out any element not in provided indices.

  • indices (Array, Required)
  • Returns instance.
query.only([0, 1, 2]); // elements at indices 0-2 (first 3 elements)

➜ instance.parent — Gets element's parent.

  • No Parameters
  • Returns instance.
query.parent();

➜ instance.parents — Gets element's parents.

  • No Parameters
  • Returns instance.
query.parents();

➜ instance.prev — Gets the previous element sibling of elements in last stack.

  • No Parameters
  • Returns instance.

Note: If no element exists null is returned.

f("#aside1").prev(); // <#aside0>

➜ instance.range(range) — Gets elements at specified indice range.

  • range (Array, Required)
    • 0 (Number, Default: 0) start_index: Where to start range.
    • 1 (Number, Default: -1) end_index: Where to end range.
      • -1 means to stop and include the last element. Essentially use the length of the array.
    • 2 (Number, Default: 1) step: Increase step by.
  • Returns instance.
query.range([0, -1, 2]); // even   
query.range([1, -1, 2]); // odd    
query.range([0, -1, 1]); // entire 
query.range([0, 3, 1]);  // < 3    
query.range([4, -1, 1]); // > 4    
query.range([0, 5, 1]);  // first 5 elements

➜ instance.siblings — Gets siblings of elements.

  • No Parameters
  • Returns instance.
query.siblings();

➜ instance.skip(indices) — Filters out element at provided indices.

  • indices (Array, Required) Indices to skip.
  • Returns instance.
query.skip([0, -1]); // filter out the first and last elements

➜ instance.state(stateType, state) — Gets elements with supplied state.

  • stateType (String, Required)
    • Possible states:
      • checked
      • selected
      • disabled
      • visible
      • empty (no elements or text nodes).
  • state (Boolean, Required)
  • Returns instance.
// Example 1: gets checked input elements
query.state("checked", true);

// Example 2: gets nonchecked input elements
query.state("checked", false);

➜ instance.tags(tag, tagN...) — Gets elements matching any of the supplied tag types.

  • tag (String, Required)
    • Note: n amount of tags may be passed.
  • Returns instance.
// Example 1: gets elements of type input or canvas
query.tags("input", "canvas");

// Example 2: gets everything BUT input or canvas elements
query.tags("!input", "!canvas");

➜ instance.textNodes — Gets the text nodes of elements in last stack.

  • No Parameters
  • Returns instance.
query.textNodes(); // filter all aside elements to get text nodes

➜ instance.getStack(index) — Return the last element stack for use.

  • index (Number, Optional, Default: 0) Index stack to return.
    • The index is optional. Omitting it will return the last element stack. However, if a previous stack is needed provide the index of the stack to return.
  • Returns instance.
query.getStack(); // use elements for whatever...

➜ instance.getElement(index) — Return the first element of the last stack for use.

  • index (Number, Optional, Default: 0) Index of element in stack to return.
    • The index is optional. Omitting it will return the first element of the last element stack. Providing an index will return the element at that index.
  • Returns instance.
f("#red", "#green").getElement()  // will return --> <#red>
f("#red", "#green").getElement(0) // will return --> <#red>
f("#red", "#green").getElement(1) // will return --> <#green>

➜ instance.concat(stack) — Combines two stacks into one.

  • stack (ElementStack, Required)
  • Returns instance.
var sidebar = f("#sidebar");
var container = f("#container");
var elements = sidebar.concat(container); // combined element collection

➜ instance.iterable — Property indicating whether the last element stack contains elements.

  • No Parameters
  • Returns Nothing.
document.addEventListener("click", function(e) {

    // cache the target element
    var target = e.target;

    // get the targets parents
    var parents = f(target).parents().getStack();
    // combine the parents with the target to return elements 
    // that contain the class green
    var filtered = f(target).concat(parents).classes("green");

    // if there are elements in the last filtered stack...
    if (filtered.iterable) {
        var delegate = filtered.getElement();
        // do something with delegate...
    }

}, false);

// ...or use it this way as it's the same thing

document.addEventListener("click", function(e) {

    // cache the target element
    var target = e.target;

    // get the targets parents
    var parents = f(target).parents().getStack();
    // combine the parents with the target to return elements 
    // that contain the class green
    var delegate = f(target).concat(parents).classes("green").getElement();

    // if there are elements in the last filtered stack...
    if (delegate) {
        // do something with delegate...
    }

}, false);

Usage

Element Filtering

General Filtering — If HTMLElements are provided to FunnelJS, its methods can be used to filter the said elements. This is very handy when using event delegation.

<div id="cont">
    <input type="text">
    <input type="password">
    <input type="hidden">
    <input type="text">
</div>
// get the element with an ID of #cont
var $cont = f("#cont").getElement();
// using the $cont element get all its ancestors and return 
// the ancestors that have the attribute type set to text
var text_inputs = f($cont).all().attrs("[type=text]").getStack();

Event Delegation Filtering — Filtering for event delegation is also made easy.

// click event listener
document.addEventListener("click", function(e) {

    // cache the target element
    var target = e.target;
    // filter element to see if it's the element we want
    var filtered = f(target).attrs("[id=cont]").getElement();

    if (filtered) { // handler logic
        console.log("Target element clicked! :)");
    }  // else ignore click 

}, false);
